# Nodeidon
Node.js daemon that watches files/directories.

# Installation
To install nodeeidon globally in your system:

`npm i -g nodeidon`

To install nodeidon as a developement dependency:

`npm i nodeidon -D`

# Usage
To start a script

`nodeidon ./examples/server.js`
